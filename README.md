## About Restoon
Restoon is a ecommerce web application. ```don't have database migration```
[restoon](https://restoon.com)

## Requirments
1. PHP >= 7.4
2. MySQL >= 5.7
3. Composer >= 1.9

## Deploy project

1. gh repo clone abrahamfast/restoon.com
2. cd restoon
3. composer install
4. cp .env.example .env
5. vim .env
